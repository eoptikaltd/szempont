# Szempont felhasználói kézikönyv

Feladat-alapú receptek a bolti csapatnak ("Hogyan..." fejezetek, R20
protokoll). Minden elkészült képernyővel bővül — nem hullámzáráskor,
hanem folyamatosan. A képernyőképek a helyben futó alkalmazásból készülnek
(`wkhtmltoimage`, `?shot=1` nézet). Hullámzáráskor nyomtatható összefűzött
változat készül; a próbaolvasó bolti kolléga, nem az alapító.
